## Hosting a Static Website on OCI Always Free
One of the simplest ways to get started with OCI Always Free is to host a static website. With OCI Always Free, you can host a static website using the Object Storage service, which provides a highly available and durable storage solution for your website's files. To host a static website on OCI Always Free, follow these steps:
1. Create an Object Storage bucket and upload your website's files
2. Configure the bucket to serve your website's files as a static website
3. Use the OCI DNS service to configure a custom domain name for your website
